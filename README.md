# Chat App

A simple cross-platform chat application built with Flutter. This app features a real-time messaging interface with a simulated auto-reply bot to demonstrate the chat flow.

## Features
- Real-time message list with smooth scrolling
- Automatic replies from a simulated bot
- Distinct visual styles for sent and received messages
- Responsive design for mobile, tablet, desktop, and web

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Design System:** Material 3

## Getting Started

1. Ensure you have Flutter installed and configured.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app on your preferred device or emulator.
